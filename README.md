# Rover

Rover is a lightweight, authenticated internal web app for quick operational workflows.

## Features

- Secure login
- Dashboard-based project navigation
- Multiple modular tools (data utilities, notes, bookmarks, shortcuts, uploads)

## Tech Stack

- Python 3.11
- Flask
- Oracle DB backend

## Run Locally

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python app.py
```

Open: `http://localhost:8080`

## Environment

Create environment variables for database connectivity and app authentication before running.
